# Asset Directory

This folder contains user-uploaded assets for certificate generation.

## Subdirectories

- **logos/** - Company/organization logos
- **signatures/** - Signature images for instructors/officials
- **backgrounds/** - Background images or patterns for certificates

## Supported Formats

- PNG (recommended for transparency)
- JPG/JPEG
- SVG
- WebP

## Usage

Upload assets through the web interface. Assets are:
1. Stored in browser's localStorage (Base64 encoded)
2. Referenced in AI prompts for template generation
3. Embedded directly in generated certificates
